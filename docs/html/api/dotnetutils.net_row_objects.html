dotnetutils.net_row_objects module¶

class ColumnValue¶

Bases: object

A wrapper for column values with different ways to access the data and its properties.

change_value(new_value)¶
Change a value

Currently only works with CodedTokens specifically, but this function does the legwork to change a value. An example of when this might be useful is deobfuscating method names

Parameters:

new_value (object) – The new value. Must match the expected type for the column.

Raises:

  • net_exceptions.InvalidArgumentsException – Invalid new_value or internal error.

  • net_exceptions.FeatureNotImplementedException – Attempted to change a column that currently isnt supported by this method.

disassemble_method(no_save=False, original=False)¶

Obtains a MethodDisassembler object for the method. This method now accounts for changes in memory of the method code using a MD5 hash.

Parameters:

  • no_save (bool) – If Enabled, the disassembler object will not be stored in the MethodDef for later use.

  • original (bool) – If enabled, the disassembler object will return for the original method’s data before any manipulation. Best to avoid this one, going to remove it probably.

Returns:

A disassembler object that can parse the method.

Return type:

net_cil_disas.MethodDisassembler
