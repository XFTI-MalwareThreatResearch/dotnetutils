append_item(item)¶

append_tx(item)¶

Append an item within a transaction. begin_append_tx() must be called beforehand.

begin_append_tx()¶

Append by transaction is a way to quickly add a bunch of things to a heap while only having to patch the binary once. Start by calling begin_append_tx(). Then use append_tx() to append all the items, then call end_append_tx(). The stream itself will not be updated until end_append_tx() is called. Attempts to call get_item() for an appended index will result in None until end_append_tx() is called.
